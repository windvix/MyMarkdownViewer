# Simple Markdown Viewer

This is a small demo document for showing the reading style of the app.

The layout is intentionally quiet: comfortable line height, focused content width, clear headings, and code blocks that feel readable instead of heavy.

## Why It Exists

- Built for reading Markdown, not editing it
- Clean spacing inspired by modern AI chat interfaces
- Works well for notes, README files, plans, and documentation
- Opens local `.md`, `.markdown`, and `.txt` files

> A good viewer should get out of the way and let the content become easy to read.

## Code Example

```js
function greet(name) {
  return `Hello, ${name}.`;
}

console.log(greet("Markdown"));
```
